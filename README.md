# KiAvenir
Prévoir l'avenir avec KiAvenir !

## Description
KiAvenir est une application web qui permet de gérer un agenda et de prévoir des événements futurs.

## Installation
```bash
npm install && npm start 
```

## Fonctionnalités
- Créer un compte
- Se connecter / Se déconnecter
- Modifier son profil
